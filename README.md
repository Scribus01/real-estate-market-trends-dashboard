## 🏡 Real Estate Market Trends Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard for analyzing real estate market trends. It provides insights into property pricing, market characteristics, and location-based patterns to support investment and development decisions.

## 🎯 Objectives
- Analyze property prices across different categories.
- Compare prices by bedrooms, condition, and grade.
- Identify high-value ZIP code areas.
- Track pricing trends over time.
- Highlight the top 10 most expensive properties.
- Enable interactive filtering through slicers.

## 📊 Dashboard Features
- KPI Cards:
  - Average Property Price
  - Total Listings
  - Average Bedrooms
  - Average Bathrooms
  - Average Living Area

- Visualizations:
  - Price Trend Over Time
  - Average Price by Bedrooms
  - Average Price by Grade
  - Average Price by Condition
  - Properties by Floors
  - Waterfront vs Non-Waterfront Distribution
  - Average Price by ZIP Code (Treemap)
  - Top 10 Most Expensive Properties

## 🛠️ Tools Used
- Power BI
- DAX
- Microsoft Excel / CSV
- Kaggle Dataset

## 📁 Dataset
The dataset contains property information such as:
- Price
- Bedrooms
- Bathrooms
- Square Footage
- Floors
- Grade
- Condition
- ZIP Code
- Latitude & Longitude
- Year Built
- Year Renovated

## 📈 Key Insights
- Property prices generally increase with the number of bedrooms.
- Higher-grade homes command significantly higher average prices.
- Certain ZIP codes consistently have premium property values.
- Waterfront properties represent a small share of listings but tend to be more valuable.
- Larger living areas are associated with higher property prices.

## 🚀 How to Use
1. Download the `.pbix` file.
2. Open it in Power BI Desktop.
3. Interact with the slicers to explore trends by property characteristics and location.
